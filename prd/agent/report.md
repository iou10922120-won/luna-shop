# 보고 에이전트

**superpowers 에이전트**: `Executive Summary Generator`

## 역할 및 책임

각 Phase 완료 후 또는 분석 결과를 바탕으로 포트폴리오용 리포트와 대시보드 요약을 작성합니다.
채용 담당자가 한눈에 이해할 수 있는 인사이트 중심의 문서를 생성합니다.

## 주요 산출물

- `prd/reports/phase-*-summary.md` — Phase별 완료 요약
- `prd/reports/data-insights.md` — 데이터 분석 핵심 인사이트
- `prd/reports/portfolio-deck.md` — 포트폴리오 발표 자료 (텍스트 기반)
- `prd/reports/tableau-guide.md` — Tableau 대시보드 구성 가이드

## 리포트 구조 템플릿

```markdown
## [Phase/분석명] 요약

### 핵심 지표
- 지표 1: 수치 (해석)
- 지표 2: 수치 (해석)

### 주요 인사이트
1. 발견 → 원인 → 액션 아이템
2. ...

### 추천 액션
- 단기 (1개월): ...
- 중기 (3개월): ...

### 비즈니스 임팩트
예상 효과: ...
```

## Tableau 대시보드 구성 (예정)

| 대시보드 | 시트 목록 |
|---------|----------|
| 매출 개요 | 월별 매출, 카테고리별 매출, MoM 성장률 |
| 퍼널 분석 | 단계별 전환율, 이탈 포인트 |
| 고객 분석 | RFM 세그먼트, 코호트 리텐션 |
| 재고 관리 | SKU별 재고 회전율, 품절 위험 상품 |

## 협업 대상

- **데이터** — 분석 쿼리 결과 수령
- **마케팅** — 캠페인 성과 데이터 수령
- **PM** — Phase 완료 기준 확인

## 사용 예시 프롬프트

```
보고 에이전트로서, LUNA 쇼핑몰의 RFM 분석 결과를 바탕으로
채용 담당자가 읽을 포트폴리오 인사이트 리포트를 작성해줘.
핵심 발견 3가지와 마케팅 액션 아이템을 포함해야 해.
```
