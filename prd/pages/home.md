# 홈페이지 (/)

## 기능 목록
1. 히어로 배너 — 브랜드 메시지 + 메인 CTA ("제품 보러가기")
2. 베스트셀러 섹션 — 상위 판매 4종 카드
3. 브랜드 스토리 섹션 — "달빛처럼, 투명하게" + 성분 투명성 소개
4. 카테고리 바로가기 — 5개 카테고리 아이콘
5. 신상품/프로모션 배너 — 시즌 캠페인 영역

## 유저 스토리
- 방문자로서, 홈에 진입했을 때 LUNA가 어떤 브랜드인지 3초 내에 파악하고 싶다
- 방문자로서, 인기 제품을 빠르게 확인하고 클릭해서 상세 페이지로 이동하고 싶다
- 재방문 유저로서, 새로운 프로모션이나 신제품이 있는지 확인하고 싶다

## 데이터 흐름

| 동작 | 읽기 (SELECT) | 쓰기 (INSERT) |
|------|-------------|--------------|
| 베스트셀러 로드 | products (ORDER BY 판매량 DESC LIMIT 4) | — |
| 카테고리 로드 | categories | — |
| 페이지 진입 | — | sessions (page_view) |

## 분석 이벤트

| 이벤트 | 트리거 | 속성 | 분석 연계 |
|--------|--------|------|----------|
| `session_start` | 첫 진입 시 | utm_source, utm_medium, utm_campaign, device_type | 트래픽 분석, 채널 귀속 |
| `page_view` | 페이지 로드 | page_path: "/" | 바운스율, 세션 시간 |
| `product_view` | 베스트셀러 상품 클릭 | product_id, product_name, category, position | 퍼널 1단계 (홈→상품 조회) |
| `product_list_view` | 카테고리 바로가기 클릭 | category | 카테고리 관심도 |

## 분석 연계
- **트래픽 분석**: 홈 유입 채널별 세션 수, 바운스율 비교
- **퍼널 진입**: 홈에서 제품 상세로 이동하는 비율 = 퍼널 1단계 전환율
- **배너 효과**: 히어로 배너 CTA 클릭률 → 캠페인별 비교

## 엣지 케이스
- 상품이 4개 미만일 때: 있는 만큼만 표시
- 이미지 로딩 실패: placeholder 이미지 표시
- 비로그인 유저: 장바구니 아이콘에 배지 없음
