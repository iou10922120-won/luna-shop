# 장바구니 (/cart)

## 기능 목록
1. 장바구니 상품 목록 (이미지, 이름, 가격, 수량)
2. 수량 변경 (+/-) 및 삭제
3. 쿠폰 코드 입력 및 적용
4. 주문 요약 (상품 금액, 할인, 배송비, 총액)
5. 결제하기 CTA

## 유저 스토리
- 구매자로서, 장바구니에 담은 상품을 확인하고 수량을 조정하고 싶다
- 가성비 유저(이하은)로서, 쿠폰을 적용해서 할인된 가격을 확인하고 싶다
- 구매자로서, 무료 배송까지 얼마나 남았는지 알고 싶다

## 데이터 흐름

| 동작 | 읽기 (SELECT) | 쓰기 (INSERT/UPDATE/DELETE) |
|------|-------------|---------------------------|
| 장바구니 로드 | cart_items JOIN products WHERE user_id = ? | — |
| 수량 변경 | — | cart_items UPDATE quantity |
| 상품 삭제 | — | cart_items DELETE |
| 쿠폰 적용 | 쿠폰 검증 (하드코딩 or DB) | — |
| 재고 실시간 확인 | products.stock_quantity | — |
| 페이지 진입 | — | sessions (cart_view) |

### 배송비 정책

|----------|--------|
| 50,000원 이상 | 무료 |

|------|------|------|

## 분석 이벤트

| 이벤트 | 트리거 | 속성 | 분석 연계 |
|--------|--------|------|----------|
| `page_view` | 페이지 로드 | page_path: "/cart" | 세션 흐름 |
| `cart_view` | 페이지 로드 | cart_total, item_count, items[] | **장바구니 이탈 분석** |
| `remove_from_cart` | 상품 삭제 | product_id, product_name, category, price | 이탈 상품 분석 |
| `coupon_apply` | 쿠폰 적용 성공 | coupon_code, discount_amount, cart_total_after | 쿠폰 효과 분석 |
| `checkout_start` | 결제하기 클릭 | cart_total, item_count | **퍼널 4단계** (결제 시작) |

## 분석 연계
- **장바구니 이탈 분석**: cart_view 후 checkout_start 미발생 세션 → 이탈 원인 분석
  - 이탈 시 장바구니 금액대 분포 → 가격 저항선 파악
  - 이탈 시 담긴 상품 조합 → 어떤 상품 조합에서 이탈이 높은지
- **쿠폰 효과**: 쿠폰 적용 세션 vs 미적용 세션의 결제 전환율 비교
- **무료배송 효과**: 무료배송 기준 근처 주문의 추가 상품 담기 비율
- **삭제 상품 분석**: 자주 삭제되는 상품 → 가격/품질 이슈 식별

## 엣지 케이스
- 장바구니 비어있음: "장바구니가 비어있어요" + [쇼핑하러 가기] 버튼
- 장바구니 상품 품절: "품절" 표시 + 결제 금액에서 제외 + 삭제 유도
- 쿠폰 코드 오류: "유효하지 않은 쿠폰입니다" 인라인 에러
- 쿠폰 중복 적용: 1개만 허용, 이미 적용된 쿠폰 교체 여부 확인
- 수량 변경 시 재고 초과: "재고가 N개 남았습니다" 토스트, 최대 수량으로 자동 조정
