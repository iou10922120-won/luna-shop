# 제품 목록 (/products)

## 기능 목록
1. 전체 상품 그리드 표시 (카드형)
2. 카테고리 필터 (클렌저/토너/세럼/크림/선크림/전체)
3. 정렬 (인기순/가격낮은순/가격높은순/신상품순/리뷰많은순)
4. 페이지네이션 (12개씩)
5. 할인 상품 배지 표시

## 유저 스토리
- 방문자로서, 전체 제품을 한눈에 보고 관심 있는 카테고리만 필터링하고 싶다
- 가격에 민감한 유저(이하은)로서, 가격 낮은 순으로 정렬해서 가성비 좋은 제품을 찾고 싶다
- 재방문 유저로서, 신상품이 나왔는지 빠르게 확인하고 싶다

상품 카드 구성:
- 이미지 (할인/신상품 배지 오버레이)
- 상품명
- 가격 (할인 시: ~~정가~~ → 할인가)
- 평균 평점 + 리뷰 수

## 데이터 흐름

| 동작 | 읽기 (SELECT) | 쓰기 (INSERT) |
|------|-------------|--------------|
| 상품 목록 로드 | products JOIN categories, 평균 평점(reviews) | — |
| 카테고리 필터 | products WHERE category_id = ? | — |
| 정렬 변경 | products ORDER BY (price/created_at/review_count) | — |
| 페이지 진입 | — | sessions (page_view, product_list_view) |
| 필터 사용 | — | sessions (product_list_filter) |

### 정렬 기준 상세
| 정렬 옵션 | SQL 기준 |
|----------|---------|
| 인기순 | ORDER BY 주문 건수 (order_items COUNT) DESC |
| 가격 낮은순 | ORDER BY COALESCE(sale_price, price) ASC |
| 가격 높은순 | ORDER BY COALESCE(sale_price, price) DESC |
| 신상품순 | ORDER BY created_at DESC |
| 리뷰 많은순 | ORDER BY 리뷰 수 (reviews COUNT) DESC |

## 분석 이벤트

| 이벤트 | 트리거 | 속성 | 분석 연계 |
|--------|--------|------|----------|
| `page_view` | 페이지 로드 | page_path: "/products" | 세션 흐름 |
| `product_list_view` | 목록 표시 | category, sort_by, item_count, page_number | 카테고리별 관심도 |
| `product_list_filter` | 필터/정렬 변경 | filter_type (category/sort), filter_value | 필터 사용 패턴 |
| `product_view` | 상품 카드 클릭 | product_id, product_name, category, position, price | 퍼널 (목록→상세 전환) |

## 분석 연계
- **카테고리별 관심도**: 어떤 카테고리 필터가 가장 많이 사용되는지
- **정렬 패턴**: 가격순 vs 인기순 — 타겟 고객의 구매 결정 기준 파악
- **목록→상세 전환율**: 카드 위치(position)별 클릭률 → 상품 노출 최적화
- **페이지별 이탈률**: 페이지네이션 2페이지 이상 가는 비율 → 상품 수/정렬 개선

## 엣지 케이스
- 카테고리 내 상품 0개: "해당 카테고리에 상품이 없습니다" 표시
- 할인가가 정가와 같을 때: 할인 배지 미표시
- 이미지 로딩 실패: placeholder 표시
- 재고 0인 상품: "품절" 배지 + 카드 비활성화(클릭은 가능, 장바구니 추가 불가)
