# 제품 상세 (/products/[id])

## 기능 목록
1. 제품 이미지 (메인 + 갤러리)
2. 제품 정보 (이름, 가격, 간략 설명)
3. 성분 투명성 카드 — LUNA 핵심 차별점
4. 수량 선택 + 장바구니 추가
5. 리뷰 목록 + 평균 평점
6. 추천 상품 (같은 카테고리)

## 유저 스토리
- 성분 중시 유저(김서연)로서, 각 성분의 역할과 안전 등급을 확인하고 구매를 결정하고 싶다
- 트렌드 유저(이하은)로서, 리뷰와 평점을 확인하고 빠르게 장바구니에 담고 싶다
- 민감 피부 유저(박지현)로서, 자극 성분이 없는지 한눈에 확인하고 싶다

## 데이터 흐름

| 동작 | 읽기 (SELECT) | 쓰기 (INSERT/UPDATE) |
|------|-------------|---------------------|
| 제품 정보 로드 | products WHERE id = ? | — |
| 성분 정보 | products.ingredients | — |
| 리뷰 로드 | reviews WHERE product_id = ? ORDER BY created_at DESC | — |
| 추천 상품 | products WHERE category_id = ? AND id != ? LIMIT 3 | — |
| 장바구니 추가 | — | cart_items INSERT (user_id, product_id, quantity) |
| 재고 확인 | products.stock_quantity | — |
| 페이지 진입 | — | sessions (product_view) |
| 성분 클릭 | — | sessions (ingredient_view) |

## 분석 이벤트

| 이벤트 | 트리거 | 속성 | 분석 연계 |
|--------|--------|------|----------|
| `page_view` | 페이지 로드 | page_path: "/products/[id]" | 세션 흐름 |
| `product_view` | 페이지 로드 | product_id, product_name, category, price, sale_price | **퍼널 2단계** (상품 조회) |
| `ingredient_view` | 성분 정보 섹션 클릭/스크롤 | product_id, action (expand/collapse) | 성분 관심도 분석 |
| `add_to_cart` | 장바구니 담기 클릭 | product_id, product_name, category, price, quantity, cart_total | **퍼널 3단계** (장바구니) |

## 분석 연계
- **퍼널 핵심**: 조회→장바구니 전환율 = 이 페이지의 KPI (벤치마크 15~25%)
- **성분 관심도**: `ingredient_view` 이벤트 발생 비율 → 성분 중시 유저 세그먼트 판별
- **가격 민감도**: 할인 상품 vs 정가 상품의 장바구니 추가율 비교
- **리뷰 효과**: 리뷰 스크롤 후 장바구니 추가 vs 리뷰 미확인 → 리뷰가 전환에 미치는 영향
- **추천 상품 효과**: 추천 상품 클릭률 → 교차 판매 기회

## 엣지 케이스
- 재고 0: "품절" 표시, 장바구니 버튼 비활성화, "재입고 알림" 대체 표시
- 리뷰 0개: "아직 리뷰가 없어요. 첫 리뷰를 남겨주세요!" 표시
- 성분 목록 없음: 성분 섹션 숨김 (데이터 무결성 이슈)
- 비로그인 장바구니 추가: 로컬 스토리지 저장 → 로그인 후 DB 동기화
- 할인 종료: sale_price null → 정가만 표시, SALE 배지 제거
