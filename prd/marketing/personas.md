## 페르소나 1: 핵심 타겟

### 김서연 (28세) — "성분 하나까지 확인하고 사는 사람"

| 항목 | 내용 |
|------|------|
| **나이** | 28세 |
| **직업** | IT 기업 UX 디자이너 (연차 4년) |
| **소득** | 연 4,200만 원 (월 실수령 약 290만 원) |
| **거주지** | 서울 마포구 원룸 |
| **MBTI** | INFJ |

#### 라이프스타일 & 가치관
- 채식은 아니지만 "가능한 범위에서" 비건 실천 (플렉시테리언)
- 환경 이슈에 민감. 텀블러·에코백 일상화, 제로웨이스트샵 이용
- 주말에 요가·필라테스, 평일 저녁 넷플릭스·책 읽기
- "내가 쓰는 돈이 어디로 가는지 알고 싶다"는 가치 소비 실천자
- 미니멀한 인테리어, 심플한 패션 선호

#### 스킨케어 루틴
| 아침 | 저녁 |
|------|------|
| 수분 클렌저 | 오일 클렌저 → 폼 클렌저 |
| 토너 패드 | 토너 |
| 세럼 (비타민C) | 세럼 (나이아신아마이드) |
| 수분 크림 | 나이트 크림 |
| 선크림 (필수) | — |

#### 화장품 소비 패턴
- **월 지출**: 5~7만 원 (스킨케어 위주)
- **구매 주기**: 2~3개월에 한 번, 시즌 전환기 집중
- **구매 채널**: 화해 앱 성분 확인 → 올리브영 체험 → 자사몰/쿠팡 구매
- **선호 가격대**: 2~4만 원 (세럼은 5만 원까지 OK)

#### Pain Points
1. "비건이라고 써놓고 실제로는 동물 유래 성분이 포함된 경우가 너무 많아"
2. "전성분 목록을 봐도 뭐가 좋고 나쁜지 한눈에 모르겠어"
3. "예쁜 디자인의 비건 화장품이 거의 없어. 왜 다 갈색·크라프트지 느낌이야?"
4. "좋은 제품 찾으면 주변에 추천하고 싶은데, 공유할 만한 콘텐츠가 없어"

#### 구매 결정 요인 (Top 3)
1. **성분 안전성** — EWG 1~2등급, 비건 인증 유무
2. **실사용 리뷰** — 화해 4.0 이상, 인스타 실사용 후기
3. **브랜드 가치관** — 동물실험 여부, 패키징 지속가능성

#### 정보 수집 채널
- 📱 화해 앱 (성분 분석 필수)
- 📷 인스타그램 (#비건뷰티 #클린뷰티 탐색)
- ▶️ 유튜브 (뷰티 유튜버 성분 리뷰)
- 💬 네이버 카페 "화장품 성분 연구소"
- 🛒 올리브영 앱 (리뷰 + 클린뷰티 카테고리)

#### LUNA에 기대하는 것
- 전 성분 원산지·인증 정보를 자사몰에서 투명하게 확인
- 내 피부 타입에 맞는 제품 추천
- 비건이지만 세련된 패키징과 브랜드 경험

> **"성분표가 곧 그 브랜드의 양심이라고 생각해요."**

## 페르소나 2: 확장 타겟

### 이하은 (24세) — "요즘 다 비건이래, 나도 써볼까?"

| 항목 | 내용 |
|------|------|
| **나이** | 24세 |
| **직업** | 대학교 4학년 (마케팅 전공) + 카페 아르바이트 |
| **소득** | 월 80~120만 원 (아르바이트 + 용돈) |
| **거주지** | 서울 관악구 (자취) |
| **MBTI** | ENFP |

#### 라이프스타일 & 가치관
- 트렌드에 민감. 인스타·틱톡에서 본 제품 바로 검색
- 가성비를 중시하지만 "좋아 보이는 것"에는 투자
- 친구들과 올리브영 들르는 게 일상. "오늘 뭐 샀어?" 공유
- 비건에 대한 확고한 신념보다는 "좋은 거라면 써보지"라는 태도
- SNS에 뷰티 루틴·신제품 언박싱 자주 공유

#### 스킨케어 루틴
| 아침 | 저녁 |
|------|------|
| 물 세안 or 약산성 폼 | 클렌징 워터 → 폼 |
| 토너 | 토너 |
| (가끔) 세럼 | 수분 크림 |
| 선크림 | 스팟 패치 (트러블 시) |

#### 화장품 소비 패턴
- **월 지출**: 2~4만 원 (소량 다품종)
- **구매 주기**: 1~2개월, 새 제품 시도 빈번
- **구매 채널**: 올리브영 오프라인 → 쿠팡 최저가 비교 → 무신사뷰티
- **선호 가격대**: 1~2.5만 원 (세럼 3만 원 한계)

#### Pain Points
1. "비건 화장품 = 비싸다는 편견이 있어서 시도하기 망설여져"
2. "올리브영에서 비건 코너 구경하는데, 다 비슷비슷해 보여"
3. "피부가 예민해져서 순한 거 찾는데, 뭘 골라야 할지 모르겠어"
4. "인스타에서 예쁜 제품 보면 사고 싶은데, 비건 제품은 디자인이 촌스러워"

#### 구매 결정 요인 (Top 3)
1. **가성비** — 가격 대비 용량, 효과
2. **디자인/패키징** — SNS에 올릴 만한 비주얼
3. **또래 리뷰** — 유튜버/인플루언서 추천, 친구 추천

#### 정보 수집 채널
- 📱 틱톡 (뷰티 숏폼, #추천템)
- 📷 인스타그램 (인플루언서 리뷰, 브랜드 피드)
- ▶️ 유튜브 (올영세일 추천템, GRWM)
- 🛒 올리브영 앱 (랭킹, 리뷰)
- 💬 에브리타임 (대학생 커뮤니티 추천)

#### LUNA에 기대하는 것
- 2만 원대의 입문용 비건 스킨케어 제품
- SNS에 올리고 싶은 감각적인 패키징
- "비건이 특별한 게 아니라 당연한 거"라는 캐주얼한 브랜드 톤

> **"예쁘고, 순하고, 비싸지 않으면 비건이든 아니든 상관없어요. 근데 비건이면 더 좋지!"**

## 페르소나 3: 잠재 타겟

### 박지현 (33세) — "아이에게 안전한 거라면 나도 쓸래"

| 항목 | 내용 |
|------|------|
| **나이** | 33세 |
| **직업** | 중견기업 인사팀 대리 (육아휴직 복귀 1년차) |
| **소득** | 가구 연 7,500만 원 (맞벌이) |
| **거주지** | 경기 성남시 (아파트, 남편+아이 18개월) |
| **MBTI** | ISTJ |

#### 라이프스타일 & 가치관
- 출산 후 피부가 급격히 민감해짐 (마스크+스트레스+호르몬 변화)
- "아이에게 닿을 수 있으니 안전한 성분이 최우선"
- 시간이 없어 루틴은 간결하게, 대신 확실한 효과 원함
- 맘카페에서 정보 수집. 검증된 제품만 구매하는 보수적 소비
- 육아용품도 성분 꼼꼼히 확인하는 습관 → 본인 화장품도 같은 기준 적용

#### 스킨케어 루틴
| 아침 | 저녁 |
|------|------|
| 물 세안 | 저자극 클렌저 |
| 올인원 크림 or 토너+크림 | 토너 → 크림 |
| 선크림 (외출 시) | (주 2회) 수면팩 |

#### 화장품 소비 패턴
- **월 지출**: 3~5만 원 (본인 + 아이 겸용 제품 선호)
- **구매 주기**: 3~4개월 (한번 정하면 재구매)
- **구매 채널**: 맘카페 추천 → 네이버 쇼핑 최저가 → 쿠팡 로켓배송
- **선호 가격대**: 2~4만 원 (효과 확실하면 5만 원 OK)

#### Pain Points
1. "출산 후 쓰던 화장품이 다 안 맞아. 뭘 발라도 따갑고 올라와"
2. "아이가 내 얼굴에 손을 대니까 유해 성분이 걱정돼"
3. "시간이 없어서 루틴을 줄이고 싶은데, 올인원은 효과가 약해"
4. "민감 피부용 제품은 대부분 비싸거나, 디자인이 병원 느낌이야"

#### 구매 결정 요인 (Top 3)
1. **성분 안전성** — 아이 접촉 안전, 무향/무색소
2. **효능 검증** — 맘카페 리뷰, 피부과 추천
3. **간편함** — 스텝 최소화, 올인원 or 멀티 기능

#### 정보 수집 채널
- 💬 맘카페 (네이버 카페: "맘스홀릭베이비", "워킹맘")
- 🔍 네이버 블로그 (민감 피부 화장품 추천 검색)
- 📱 화해 앱 (성분 등급 확인)
- 👩‍⚕️ 피부과/소아과 추천
- 🛒 쿠팡 (로켓배송 재구매)

#### LUNA에 기대하는 것
- 민감 피부 + 아이 접촉 안전한 성분 (무향, 저자극 라인)
- "이 제품은 이 성분 때문에 안전합니다"라는 명확한 근거
- 간편한 루틴 제안 (2~3스텝으로 끝나는 세트)

> **"비건이어서가 아니라 성분이 안전해서 선택하는 거예요. 우리 아이한테도 닿을 수 있으니까요."**
