## 브랜드 스토리

### 시작

밤하늘의 달은 화려하지 않습니다.
조용히 빛나며, 밤새 세상을 비춥니다.

LUNA는 그 달빛처럼,
과장 없이 정직하게, 당신의 피부를 돌보는 브랜드입니다.

### 왜 LUNA인가

우리는 화장품 성분표 뒤에 숨겨진 것들이 궁금했습니다.

"비건이라고 쓰여 있지만, 정말 비건일까?"
"클린뷰티라고 하는데, 기준이 뭘까?"
"이 성분은 어디서 왔고, 왜 넣었을까?"

이 질문에 떳떳하게 답할 수 있는 브랜드를 만들고 싶었습니다.
그래서 LUNA는 모든 것을 보여주기로 했습니다.

### 우리의 약속

**모든 성분을 공개합니다.**
전성분 목록만이 아닙니다. 각 성분의 원산지, 역할, 인증 정보까지.
당신이 피부에 바르는 것에 대해, 우리가 알고 있는 전부를 공유합니다.

**모든 제품은 비건입니다.**
일부가 아닌 전 제품. PETA와 한국비건인증원의 이중 인증.
동물에게 해를 끼치지 않는 것은 선택이 아닌 기본입니다.

**포장도 피부만큼 신경 씁니다.**
재활용 가능한 소재, 최소한의 포장, 리필 시스템.
예쁜 쓰레기를 만들지 않겠습니다.

### 달빛 아래에서

밤마다 당신의 피부는 회복합니다.
LUNA는 그 시간을 함께합니다.

투명하게, 정직하게, 아름답게.
달빛처럼.

## 슬로건

### 메인 슬로건

> **"달빛처럼, 투명하게."**

빛나지만 과하지 않은 달의 이미지 + 성분 투명성의 핵심 가치를 결합.
간결하고 기억하기 쉬우며, 브랜드 네임(LUNA=달)과 자연스럽게 연결.

### 서브 슬로건 (용도별)

| 용도 | 슬로건 | 사용 맥락 |
|------|--------|----------|
| 브랜드 태그라인 | 달빛처럼, 투명하게. | 로고 하단, 자사몰 헤더 |
| 성분 투명성 | 숨기지 않으니까, LUNA. | 성분 공개 페이지, 제품 상세 |
| 비건 가치 | 피부에도, 지구에도, 순하게. | 비건 인증 설명, SNS |
| 제품 체험 | 오늘 밤, 달빛 루틴. | 나이트 케어 라인, 캠페인 |
| 첫 구매 유도 | 성분표를 읽어본 적 있나요? | 광고 카피, 랜딩 페이지 |

## 핵심 메시지 프레임워크

### 메시지 계층 구조

```
Level 1 (브랜드 존재 이유)
"모든 성분을 투명하게 공개하는 비건 스킨케어"

  └── Level 2 (핵심 가치 3가지)
      ├── 투명성: "성분 하나까지 숨기지 않습니다"
      ├── 비건: "전 제품 비건 인증, 예외 없이"
      └── 아름다움: "순하면서도 세련되게"

        └── Level 3 (증거/Proof Points)
            ├── PETA + 한국비건인증원 이중 인증
            ├── 성분별 원산지·역할·등급 공개
            ├── 파라벤·인공향·황산염 0%
            ├── 리필 시스템 + PCR 패키징
            └── 피부 진단 기반 맞춤 추천
```

### 톤 & 보이스 가이드라인

| DO | DON'T |
|----|-------|
| "이 성분은 ~에서 왔어요" | "최고의 원료만을 엄선" |
| "비건 인증을 받았습니다" | "자연에서 온 순수한 에너지" |
| "솔직히 말하면" | "혁신적인", "획기적인" |
| 구체적인 숫자·인증 | 모호한 형용사 나열 |
| 대화하듯 편안하게 | 강의하듯 권위적으로 |
| "우리도 고민했어요" | "우리가 정답입니다" |

### 채널별 톤 조정

| 채널 | 톤 | 예시 |
|------|-----|------|
| **자사몰** | 신뢰감 + 상세 | "이 세럼의 나이아신아마이드 5%는 독일 BASF에서 공급받습니다." |
| **인스타그램** | 친근 + 감성 | "오늘 밤 루틴 끝. 달빛 크림 바르고 자는 게 제일 좋아 🌙" |
| **유튜브** | 교육 + 솔직 | "시카 크림, 진짜 효과 있을까? 성분부터 까봤습니다." |
| **카카오** | 실용 + 혜택 | "서연님, 토너 리필 시기예요! 지금 주문 시 20% 할인" |

|------|------|

### 개발 시 적용 지침 (web/ 구현 참고)
- 폰트: 한글 Pretendard + 영문 Inter (산세리프)
- 배경: `#F8F6F3` 아이보리 기본
- CTA 버튼: `#2C3E6B` 네이비 + 흰 텍스트
- 이미지 톤: 달빛/자연광, 따뜻한 화이트 톤, 과도한 보정 없이
- 여백: 넉넉하게 (최소 24px)
