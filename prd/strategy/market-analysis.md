# 국내 비건 화장품 시장 분석

> LUNA 프로젝트 — Phase 1: 브랜드 전략
> 작성일: 2026-03-20

---

## 1. 시장 규모 및 성장률

### 글로벌 비건 화장품 시장

| 지표 | 수치 |
|------|------|
| 2024년 시장 규모 | 약 168억 달러 (USD) |
| 연평균 성장률 (CAGR) | 8.5~9.5% (2024~2030) |
| 2030년 전망 | 약 280억 달러 |

- 북미·유럽이 시장을 주도하나, 아시아태평양 지역이 가장 빠르게 성장 중
- 동물실험 금지 입법(EU Cosmetics Regulation, 한국 화장품법 개정)이 시장 확대를 가속

### 국내 비건/클린뷰티 시장

| 지표 | 수치 |
|------|------|
| 2024년 추정 규모 | 약 7,200~8,000억 원 |
| 2025년 추정 규모 | 약 8,000~8,800억 원 |
| 연평균 성장률 | 12~15% (국내 화장품 시장 대비 2배) |
| 전체 화장품 시장 내 비중 | 약 5~7% (빠르게 증가 중) |

- 2017년 한국 동물실험 금지법 이후 비건 인증 브랜드 급증
- 올리브영 '클린뷰티' 코너 신설 (2020~) → 소비자 접근성 대폭 향상

## 2. 글로벌 트렌드

### 클린뷰티 (Clean Beauty)
- 유해 성분 배제 (파라벤, 황산염, 인공향, 미네랄오일 등 20여 종)
- "성분 전성분 공개"가 기본이 된 시대
- FDA/EU의 성분 규제 강화 흐름

### 비건뷰티 (Vegan Beauty)
- 동물 유래 성분 완전 배제 (밀랍, 라놀린, 카르민 등)
- PETA, Leaping Bunny, 한국비건인증원 등 제3자 인증 필수화
- "비건 = 프리미엄"이 아닌 "비건 = 기본"으로 인식 전환 중

### 지속가능 패키징 (Sustainable Packaging)
- 리필 시스템, PCR(재활용 플라스틱), 종이 패키징, 고체 제형
- 포장재 과대 사용에 대한 MZ세대 반감 증가
- 유럽 Green Deal 패키징 규제 → 글로벌 기준 상향

### 스킨티피케이션 (Skinification)
- 스킨케어 성분 중심 사고가 메이크업·헤어·바디로 확장
- 나이아신아마이드, 세라마이드, 히알루론산 등 성분명이 마케팅 키워드
- 소비자가 "성분표를 읽는" 시대

### AI/개인화
- 피부 진단 AI → 맞춤 추천
- D2C 브랜드의 차별화 무기로 부상
- 고객 데이터 기반 제품 개발 사이클 단축

## 3. 국내 트렌드

### 소비자 인식 변화 3단계

```
1단계 (2017~2019): "동물실험 안 한 거면 되지 않나?"
  → 동물실험 금지법 시행, 초기 인식 형성

2단계 (2020~2022): "성분도 중요하지"
  → 화해 앱 500만 DL 돌파, 올리브영 클린뷰티 코너
  → 성분 분석이 구매 전 기본 행위로 자리잡음

3단계 (2023~현재): "브랜드 가치도 봐야지"
  → 비건 인증 + 지속가능 패키징 + 브랜드 스토리까지 종합 평가
  → 가치 소비 (미닝아웃) 확산
```

### 채널 동향

| 채널 | 동향 |
|------|------|
| **올리브영** | 클린뷰티 전용 코너, 입점 기준에 성분 심사 강화. 인디 브랜드 런칭패드 역할 |
| **무신사뷰티** | 2030 남녀 타겟, 트렌드 큐레이션 강점. 신생 브랜드에 기회 |
| **쿠팡** | 로켓배송 비건 제품 카테고리 확대. 가격 경쟁 심화 |
| **D2C 자사몰** | 브랜드 스토리텔링 + 데이터 수집 가능. CAC 부담 존재 |
| **카카오/네이버** | 선물하기, 라이브커머스 통한 발견형 구매 증가 |

### MZ세대 구매 행태

- **정보 탐색**: 화해 앱 성분 확인 → 인스타/유튜브 리뷰 → 올리브영 체험 → 온라인 구매
- **구매 주기**: 스킨케어 기초 2~3개월, 시즌 전환기 집중 구매
- **월 평균 지출**: 3~7만 원 (스킨케어 기초 기준)
- **구매 결정 요인**: 성분 > 가격 > 리뷰/평점 > 브랜드 가치 > 디자인

## 4. TAM / SAM / SOM

### TAM (Total Addressable Market)
**국내 비건/클린뷰티 화장품 전체 시장**
- 약 **8,000~8,800억 원** (2025년 추정)
- 스킨케어 + 메이크업 + 바디 + 헤어 포함

### SAM (Serviceable Addressable Market)
**온라인 D2C/이커머스 채널의 비건 스킨케어**
- 전체 비건 화장품의 약 30~35%가 온라인 구매
- 스킨케어가 전체의 약 55~60%
- 약 **2,400~2,900억 원**

### SOM (Serviceable Obtainable Market)
**LUNA가 1~3년 내 현실적으로 확보 가능한 매출**
- 인디 D2C 브랜드 기준 온라인 시장 점유율 0.5~1.2%
- **1년차**: 약 12~15억 원 (월 1~1.2억)
- **3년차**: 약 30~36억 원 (월 2.5~3억)

```
┌─────────────────────────────────────────────┐
│              TAM: 8,000~8,800억              │
│  ┌─────────────────────────────────────┐    │
│  │        SAM: 2,400~2,900억           │    │
│  │  ┌─────────────────────────────┐    │    │
│  │  │    SOM: 12~36억 (1~3년)    │    │    │
│  │  └─────────────────────────────┘    │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

## 5. 성장 동인

### 규제 환경
- **동물실험 금지**: 한국 (2017), EU 전면 금지 → 비건 제품 개발 의무화
- **ESG 경영**: 화장품 대기업의 ESG 보고서에 비건/클린 라인 확대 명시
- **포장재 규제**: 화장품 과대포장 금지, 재활용 용이성 등급제

### 소비자 인식
- **성분 리터러시 향상**: 화해, 글로우픽 등 성분 분석 플랫폼 활성화
- **가치 소비 확산**: "내가 쓰는 제품이 환경에 미치는 영향" 고려
- **알레르기/민감성 증가**: 미세먼지, 마스크 장기착용 후 민감 피부 급증

### MZ세대 주도
- **미닝아웃(Meaning Out)**: 소비를 통한 가치 표현
- **디지털 네이티브**: 성분 비교, 리뷰 탐색이 자연스러운 소비 프로세스
- **SNS 바이럴**: 비건/클린 제품 언박싱, GRWM 콘텐츠 확산

### 기술/인프라
- **바이오 기술**: 식물 유래 대체 성분 기술 발전 (스쿠알란, 세라마이드)
- **D2C 인프라**: Shopify, Cafe24, NHN 커머스 등 자사몰 구축 비용 감소
- **데이터 분석**: 고객 행동 데이터 기반 제품 개발·마케팅 최적화

## 6. 리스크 / 위협

### 레드오션화
- 대기업 (아모레, LG생활건강)의 비건 라인 대거 출시
- 올리브영 PB "비건 에디션" 론칭
- 인디 브랜드 우후죽순 → 차별화 포인트 약화

### 그린워싱 (Greenwashing)
- 실제 비건이 아니면서 비건 마케팅하는 브랜드 증가
- 소비자 불신 → 인증의 중요성 부각
- 정부의 그린워싱 규제 강화 예고

### 가격 경쟁
- 다이소 1,000~3,000원 스킨케어 열풍
- 쿠팡 로켓배송 PB 가격 경쟁
- 프리미엄 매스 가격대(2~5만원)의 가치 정당화 필요

### 초기 트래픽/CAC 부담
- D2C 자사몰 초기 유입 확보 난이도 높음
- 인스타그램/네이버 광고 단가 지속 상승
- 평균 CAC 15,000~30,000원 (화장품 D2C 기준)

### 위협 매트릭스

| 위협 | 발생 가능성 | 영향도 | 대응 전략 |
|------|-----------|--------|----------|
| 대기업 비건 라인 확대 | 높음 | 중간 | 인디 브랜드 특유의 진정성·커뮤니티로 차별화 |
| 그린워싱 불신 확산 | 중간 | 높음 | 제3자 인증 + 전 성분 투명 공개 |
| 가격 하방 압력 | 높음 | 높음 | 성분·스토리 기반 가치 마케팅 |
| 초기 트래픽 부족 | 높음 | 높음 | SNS 커뮤니티 빌딩, 인플루언서 시딩 |
| 공급망 이슈 | 낮음 | 중간 | 복수 공급처 확보, 국내 OEM 우선 |

## 7. LUNA 전략적 시사점

### 차별화 방향
1. **성분 투명성 D2C**: 전 제품 전성분 + 원산지 + 인증 정보를 자사몰에 상세 공개
2. **커뮤니티 기반 성장**: 성분 매니아 고객과의 양방향 소통 (리뷰, Q&A, 성분 교육 콘텐츠)
3. **지속가능 패키징**: 리필 시스템 + 재생 플라스틱 + 최소 포장

### 채널 전략
```
Phase 1 (0~6개월): D2C 자사몰 집중 → 브랜드 아이덴티티 확립
Phase 2 (6~12개월): 올리브영 온라인 입점 → 트래픽 확보
Phase 3 (12~18개월): 무신사뷰티 + 카카오 선물하기 → 채널 다각화
```

### 핵심 경쟁 우위
- **데이터 드리븐 D2C**: 고객 행동 데이터 기반 제품 개발·마케팅 (← 이 프로젝트의 핵심 포트폴리오 가치)
- **진정성 있는 비건**: PETA/비건소사이어티 이중 인증 + 투명한 공급망
- **MZ 감성 브랜딩**: 심플하고 세련된 디자인 + 가치 중심 스토리텔링
