# LUNA 브랜드 포지셔닝

> LUNA 프로젝트 — Phase 1: 브랜드 전략
> 작성일: 2026-03-20

---

## 1. 포지셔닝 스테이트먼트

> **LUNA**는 **성분에 진심인 20~30대 여성**을 위한
> **전 제품 비건 인증 스킨케어 D2C 브랜드**입니다.
>
> 대기업의 그린워싱도, 인디 브랜드의 소박함도 아닌,
> **투명한 성분 정보 × 세련된 디자인 × 데이터 기반 개인화**로
> "비건 스킨케어의 새로운 기준"을 제시합니다.

---

## 2. 브랜드 아이덴티티

### 브랜드 이름: LUNA (루나)

**의미**
- 달(Luna)처럼 매일 밤 피부를 회복시키는 스킨케어
- 부드럽고 차분한 이미지 + 신비로운 아름다움
- 한국어 발음이 자연스럽고, 글로벌 확장 시 인지도 유리

### 브랜드 키워드

```
비건(Vegan) · 투명(Transparent) · 세련(Refined) · 과학적(Scientific) · 달빛(Moonlit)
```

### 브랜드 성격 (Brand Personality)
| 차원 | LUNA | 설명 |
|------|------|------|
| **톤** | 신뢰감 있되 따뜻한 | 전문적이지만 병원 느낌은 아닌 |
| **태도** | 정직하고 겸손한 | 과장 광고 없이 성분으로 말하는 |
| **감성** | 미니멀 럭셔리 | 깔끔한 디자인, 촉감 좋은 패키징 |
| **관계** | 믿을 수 있는 친구 | 성분 질문에 솔직하게 답하는 브랜드 |

## 3. 경쟁 포지셔닝 정리

### LUNA vs 경쟁사 차별화 포인트

| 경쟁 영역 | 경쟁사 방식 | LUNA 방식 |
|----------|-----------|----------|
| **성분 공개** | 전성분 목록만 나열 | 성분별 원산지·역할·인증 정보 상세 공개 |
| **비건 인증** | 일부 제품만 or 자체 주장 | 전 제품 PETA + 한국비건인증원 이중 인증 |
| **패키징** | 크라프트/갈색 (인디) or 기존 플라스틱 (대형) | 미니멀 럭셔리 + 재활용 소재 + 리필 옵션 |
| **고객 경험** | 일방적 판매 | 성분 상담 챗, 피부 진단 추천, 커뮤니티 |
| **데이터 활용** | 거의 없음 | 구매/행동 데이터 → 맞춤 추천·리텐션 마케팅 |

### 핵심 차별화 삼각형

```
         성분 투명성
            △
           / \
          /   \
         / LUNA \
        /   ★   \
       ───────────
  세련된 디자인    데이터 개인화
```

이 세 가지를 동시에 충족하는 비건 스킨케어 브랜드가 현재 부재 → LUNA의 포지션

## 4. 비즈니스 모델 캔버스 (BMC)

### Customer Segments (고객 세그먼트)
- **핵심**: 25~30세 성분 중시 직장인 여성 ("김서연")
- **확장**: 22~26세 트렌드 민감 대학생/사회초년생 ("이하은")
- **잠재**: 30~35세 민감 피부 워킹맘 ("박지현")

### Value Propositions (가치 제안)
- 전 제품 비건 인증 (PETA + 한국비건인증원)
- 업계 최고 수준의 성분 투명성 (원산지·인증·역할 공개)
- 세련된 미니멀 럭셔리 디자인
- 데이터 기반 맞춤 추천 (피부 타입 진단)
- 지속가능 패키징 (리필 시스템, PCR 소재)

### Channels (채널)
| 단계 | 채널 | 목적 |
|------|------|------|
| Phase 1 | D2C 자사몰 (luna-shop.vercel.app) | 브랜드 경험, 데이터 수집 |
| Phase 2 | 올리브영 온라인 | 트래픽 확보, 인지도 |
| Phase 3 | 무신사뷰티 + 카카오 선물하기 | 채널 다각화 |
| 상시 | 인스타그램 + 유튜브 | 콘텐츠 마케팅, 커뮤니티 |

### Customer Relationships (고객 관계)
- **성분 상담**: 챗봇/Q&A를 통한 1:1 성분 상담
- **커뮤니티**: 성분 토론, 리뷰, 스킨케어 루틴 공유
- **개인화**: 피부 진단 → 맞춤 루틴 추천
- **리텐션**: 정기 배송, 리필 할인, 적립금

### Revenue Streams (수익 흐름)
| 수익원 | 비중 (목표) | 설명 |
|--------|-----------|------|
| 제품 판매 (자사몰) | 60% | 풀라인 스킨케어 |
| 정기 구독 | 20% | 리필 정기 배송 (10% 할인) |
| 입점 채널 판매 | 15% | 올리브영, 무신사뷰티 |
| 한정판/콜라보 | 5% | 시즌 한정, 아티스트 콜라보 |

### Key Resources (핵심 자원)
- 비건 인증 제품 라인업 (OEM 파트너)
- D2C 플랫폼 (Next.js + Supabase)
- 고객 행동 데이터 + 분석 역량
- 브랜드 디자인 에셋

### Key Activities (핵심 활동)
- 제품 기획/개발 (OEM 협업)
- D2C 사이트 운영 + 데이터 분석
- 콘텐츠 마케팅 (성분 교육, 리뷰)
- 커뮤니티 빌딩

### Key Partners (핵심 파트너)
- OEM/ODM 제조사 (코스맥스, 한국콜마 등)
- 비건 인증 기관 (PETA, 한국비건인증원)
- 물류 파트너 (CJ대한통운)
- 인플루언서/뷰티 크리에이터

### Cost Structure (비용 구조)
| 비용 항목 | 비중 (초기) | 설명 |
|----------|-----------|------|
| 제품 제조 (OEM) | 35% | 원가율 30~40% |
| 마케팅/광고 | 30% | 인스타, 네이버, 인플루언서 |
| 물류/배송 | 15% | 풀필먼트, 택배비 |
| 플랫폼 운영 | 10% | 서버, 결제 수수료 |
| 인건비 | 10% | 초기 소규모 팀 |

## 5. 가격 전략

```
다이소      올리브영 PB     이니스프리     LUNA      닥터자르트     에스티로더
1~5천원     5천~1.5만원    1.5~3.5만원   2~5만원    3~5만원       5~15만원
─────────┼──────────┼──────────┼─────────┼─────────┼──────────
  매스       매스 프리미엄    미들          프리미엄 매스   프리미엄      럭셔리
```

### 가격 정당화 전략
1. "성분값" 공개 — 원료 원가의 투명한 공개
2. "인증값" 강조 — PETA + 한국비건인증원 이중 인증 비용
3. "지속가능값" — 재활용 패키징, 리필 시스템 투자
4. 리필 구매 시 20% 할인 → 지속가능성 + 재구매율 동시 확보
